Encryptor - File Encryption Tool
=================================

Requirements:
- Python 3.10.7 or newer
- Pip (Python package manager)

How to Install Python:
----------------------
1. Download Python from:
   https://www.python.org/downloads/

2. During installation, make sure to enable:
   "Add Python to PATH"

3. Verify installation:
   python --version

How to Install Pip (if missing):
--------------------------------
Most Python installations include pip. Check with:
   pip --version

If pip is missing:
   python -m ensurepip --default-pip

Required Library:
-----------------
Install the cryptography library(in cmd or pworeshell):
   pip install cryptography
or if it didn't work try:
   python -m pip install cryptography

How to Run the Application:
---------------------------
1. Extract this ZIP file.
2. Open a terminal / command prompt inside the extracted folder.
3. Run the application by typing:

   python encryptor.py

The application window will open. You can now encrypt and decrypt files securely.
